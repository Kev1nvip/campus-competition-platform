# campus-competition-platform
# 校园学术竞赛管理平台

## 项目结构
campus-competition-platform/
├── frontend/   # Vue3 + Vite 前端项目
└── backend/    # SpringBoot3 后端项目

## 技术栈
- 前端：Vue3 + Vite + Element Plus
- 后端：SpringBoot3 + PostgreSQL + Redis + RabbitMQ
- 权限：Spring Security + JWT

## 本地启动
见各子目录 README.md

## 分支规范
- main：主分支，受保护
- dev: 开发分支
- feature/xxx：功能分支

## Commit 规范
- feat：新功能
- fix：修复Bug
- docs：文档
- chore：工程配置
